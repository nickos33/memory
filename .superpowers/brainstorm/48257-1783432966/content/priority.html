Visual + Layout + Features + Polish — 全部覆盖，分优先级推进

优先级排序

按影响力和实现成本排序，建议的推进顺序：

P0 · 立即见效 · 低成本

倒计时天数显示、卡片入场动画、空状态优化、hover 微交互

P1 · 显著提升 · 中成本

整体视觉风格升级（玻璃态/毛玻璃）、卡片重新设计、字体和间距精细调校

P2 · 体验跃升 · 中高成本

拖拽排序 (dnd-kit)、一键置顶、键盘快捷键 (Cmd+N 新增, / 搜索, Esc 关闭)

P3 · 锦上添花 · 高成本

农历日期支持、农历生日自动推算、分类标签、数据导出/导入、农历倒计时小组件（macOS 通知中心）
