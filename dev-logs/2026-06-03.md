## 遇到的问题（阻塞）

- **Electron 内置 `electron` 模块无法被找到**：`import electron from 'electron/main'` 报 MODULE_NOT_FOUND
- 从 npmmirror 国内镜像下载的 Electron v28.2.0 zip 中缺少 `electron.asar` 文件
- 尝试了多种方案均未解决：
  - CJS require → 解析到 node_modules/electron 返回路径字符串
  - 删除 node_modules/electron → MODULE_NOT_FOUND（证明无内置模块）
  - ESM import electron/main → 同样 MODULE_NOT_FOUND
  - 从 /tmp（无 node_modules 干扰）运行 → 同样失败
- GitHub 官方下载被阻断（网络问题）
