# 雾面米灰小组件设计

## 目标

降低桌面小组件与桌面背景的亮度反差，弱化边框和悬浮阴影，让小组件自然融入桌面，同时保持信息可读性和现有交互。

## 视觉调整

- 面板改为低亮度暖米灰半透明背景，让桌面颜色适度透出。
- 保留背景模糊，但降低白色边框的不透明度，使边缘接近不可见。
- 将大范围悬浮阴影改为范围较小、透明度较低的贴地阴影。
- 纪念日条目底色进一步弱化，减少面板内的嵌套卡片感。
- 主文字使用暖深灰，次要文字使用低对比暖灰。
- 临近日期继续使用低饱和陶土红，保留必要的信息层级。

## 保持不变

- 小组件尺寸、位置和置顶层级不变。
- 拖动、滚动、点击打开主窗口、右键隐藏、恢复全部和刷新行为不变。
- DOM 结构、数据来源、日期计算和主界面样式不变。
- 不增加图片、在线资源或运行时依赖。

## 验证标准

- 面板不再呈现高亮米白色块。
- 边框和阴影明显弱于当前版本，但面板在常见深浅桌面上仍可辨认。
- 名称、备注和倒计时保持清晰；临近日期仍能被快速识别。
- 小组件核心 CSS 选择器保持完整。
- 自动化测试、生产渲染构建和 arm64 目录打包通过。
