# UI 设计规范

## 配色方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 主背景 | `#F5F5F5` | 浅灰底色 |
| 卡片背景 | `#FFFFFF` | 白色卡片 |
| 淡蓝-最浅 | `#E3F2FD` | 蓝色调背景 |
| 淡蓝-浅 | `#BBDEFB` | 边框/悬浮 |
| 淡蓝-中 | `#90CAF9` | 输入框聚焦 |
| 淡蓝-主色 | `#2196F3` | 主按钮/选中态 |
| 淡蓝-深 | `#1976D2` | 链接/强调文字 |
| 红色-最浅 | `#FFEBEE` | 置顶卡片渐变 |
| 红色-浅 | `#FFCDD2` | 删除按钮边框 |
| 红色-主色 | `#E53935` | 新增按钮/删除确认 |
| 红色-深 | `#D32F2F` | 按钮 hover |

## 组件规范

### 卡片 (Card)
- 圆角 12px，白底，左边框 4px
- 置顶卡片：左边框红色 + 渐变背景
- 阴影：默认 sm(0 1px 3px)，hover 时 md(0 4px 12px)
- 卡片间距：12px

### 按钮
- 主操作（新增/保存）：红色/蓝色实心
- 次要操作（编辑/删除）：透明底+彩色边框
- hover 时颜色加深，active 时缩放 0.97

### 弹窗 (Modal)
- 居中覆盖，半透明黑色遮罩
- 白色圆角面板，入场动画 slideUp 0.2s

### 搜索栏
- 白色背景，聚焦时蓝色边框+光环

### 表单
- 标签 14px 粗体，输入框 10px padding
- 事由选择：chip 样式按钮组
- 错误提示：红色 12px 文字

## 字体

- 系统默认字体：`-apple-system, BlinkMacSystemFont, "SF Pro Text"`
- 标题：28px 粗体
- 卡片人名：20px 粗体
- 卡片事由：15px 蓝色中粗
- 正文：14-15px 常规

## 间距

- 页面 padding：24px 20px
- 内容最大宽度：720px
- 组件间距：12-20px
