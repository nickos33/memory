# 需求规格说明书

## 概述

一款 macOS 桌面应用，帮助用户记录和管理各类纪念日，并在纪念日临近时发送系统通知提醒。

## 用户角色

- 普通用户：所有功能的直接使用者

## 功能需求

### F1 — 纪念日管理

| 编号 | 功能 | 描述 |
|------|------|------|
| F1.1 | 新增 | 通过表单录入纪念日：日期、人名、事由、备注 |
| F1.2 | 编辑 | 修改已录入的纪念日信息 |
| F1.3 | 删除 | 删除纪念日，删除前需确认 |
| F1.4 | 置顶 | 将纪念日卡片固定在列表顶部 |
| F1.5 | 取消置顶 | 取消已置顶的卡片 |

### F2 — 卡片展示

| 编号 | 功能 | 描述 |
|------|------|------|
| F2.1 | 卡片列表 | 所有纪念日以卡片形式纵向排列 |
| F2.2 | 日期排序 | 按纪念日日期降序排列，置顶卡片优先 |
| F2.3 | 空状态 | 无数据时显示引导提示 |
| F2.4 | 临近标识 | 当天/明天的纪念日显示特殊标记 |

### F3 — 搜索

| 编号 | 功能 | 描述 |
|------|------|------|
| F3.1 | 实时搜索 | 按人名、事由、备注实时过滤 |
| F3.2 | 无结果提示 | 搜索无匹配时显示空结果状态 |

### F4 — 提醒通知

| 编号 | 功能 | 描述 |
|------|------|------|
| F4.1 | 临近提醒 | 纪念日前 7 天、3 天、1 天、当天发送通知 |
| F4.2 | 通知点击 | 点击通知打开应用主窗口 |
| F4.3 | 定时检查 | 每天上午 9:00 自动检查 |

### F5 — 数据存储

| 编号 | 功能 | 描述 |
|------|------|------|
| F5.1 | 本地存储 | 所有数据保存在本地 JSON 文件 |
| F5.2 | 离线可用 | 无需网络连接，全部功能离线可用 |

## 非功能需求

- UI 响应时间 < 200ms
- 窗口最小尺寸 600×400px
- 数据持久化，应用重启后不丢失
